# InfluencerFinder

AI-powered influencer discovery tool with deep email scraping for TikTok & Instagram.

## Features
- 🔍 Hybrid Search Strategy (Google + Apify)
- 📧 Deep Email Extraction from Instagram Bio
- 🎯 Precise Follower & Engagement Rate Filtering
- 📊 Export to CSV
- 💾 Search History & Collections

## Tech Stack
- Next.js 16 + TypeScript
- Google Custom Search API
- Apify Scraper API
- Vercel Deployment
